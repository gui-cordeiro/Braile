# 🕹️ Jogo do Braile 🕹️
## ♿ Aprenda este importante sistema de escrita tátil de forma simples e divertida! ♿


* Este jogo foi desenvolvido em Linguagem C, como forma de avaliação parcial para o trabalho final do 1º ano do Curso Técnico em Informática, ofertado pelo Instituto Federal do Paraná - Campus Colombo e apresentado no ano de 2017, com o intuito de ensinar o sistema de escrita tátil "Braile" de forma simples e lúdica para pessoas sem deficiência visual.
